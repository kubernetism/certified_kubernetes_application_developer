# Chapter 05: Pods and Namespaces

## Overview
This chapter covers Kubernetes pods and namespaces, the fundamental building blocks for application deployment and organization in Kubernetes.

## Topics Covered
- Kubernetes pods and their lifecycle
- Pod specifications and configuration
- Pod networking and communication
- Namespaces and multi-tenancy
- Pod scheduling and placement
- Pod health checks and readiness

## Learning Objectives
- Understand pod architecture and lifecycle
- Master pod configuration and specifications
- Learn namespace management and isolation
- Understand pod networking and communication
- Implement pod health checks and readiness probes
