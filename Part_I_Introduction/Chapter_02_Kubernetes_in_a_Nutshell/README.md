# Chapter 02: Kubernetes in a Nutshell

## Overview
This chapter provides a comprehensive overview of Kubernetes fundamentals, including architecture, components, and core concepts.

## Topics Covered
- Kubernetes architecture and components
- Pods, Deployments, and Services
- Namespaces and context
- Kubernetes objects and resources
- Basic Kubernetes operations

## Learning Objectives
- Understand Kubernetes architecture and master components
- Learn about core Kubernetes objects and their relationships
- Master basic Kubernetes operations and commands
- Understand Kubernetes namespaces and multi-tenancy
