# Chapter 03: Interacting with Kubernetes

## Overview
This chapter covers the various ways to interact with Kubernetes clusters, focusing on the `kubectl` command-line tool and other interaction methods.

## Topics Covered
- kubectl command-line interface
- Kubernetes YAML manifests
- Context and namespace management
- kubectl plugins and extensions
- Kubernetes dashboard and web interfaces

## Learning Objectives
- Master kubectl commands and operations
- Understand Kubernetes manifest structure and best practices
- Learn context and namespace management
- Explore various Kubernetes interaction interfaces
