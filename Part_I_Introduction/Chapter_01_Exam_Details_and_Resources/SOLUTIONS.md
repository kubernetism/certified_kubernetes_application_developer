# Chapter 01: Exam Details and Resources - Solutions

## Practice Questions - Solutions

### Question 1: Exam Structure
**Question:** What is the format and duration of the CKAD exam?
**Correct Answer:** The CKAD exam is a performance-based exam delivered online, consisting of 19-20 questions that must be completed within 2 hours.

### Question 2: Exam Domains
**Question:** What are the main domains covered in the CKAD exam and their approximate weightings?
**Correct Answer:**
- Application Design and Build (20%)
- Application Deployment (25%)
- Application Lifecycle and Management (15%)
- Services and Networking (20%)
- Observability and Application Maintenance (15%)
- Security and Configuration (5%)

### Question 3: Passing Score
**Question:** What is the passing score for the CKAD exam?
**Correct Answer:** The passing score for the CKAD exam is 72%.

### Question 4: Exam Interface
**Question:** How is the CKAD exam delivered (online, proctored, etc.)?
**Correct Answer:** The CKAD exam is delivered online via a proctored interface. Candidates use a remote browser that provides a command line and Kubernetes cluster for hands-on tasks.

### Question 5: Preparation Resources
**Question:** List three official resources recommended for CKAD exam preparation.
**Correct Answer:**
1. Certified Kubernetes Application Developer (CKAD) Exam Course
2. Kubernetes documentation (kubernetes.io)
3. Kubernetes official tutorials and guides
